## Текущая задача
**Новый дизайн-этаж «Окраина» (Outskirts): Переходная зона между Жилой и Коллекторами. Генератор + блокировка лифтов вниз.**

### 1. Глубокий контекст задачи и бизнес-логика
«Окраина» (Outskirts) — это уникальный дизайн-этаж. Он служит "шлюзом" и переходной зоной между относительно безопасными Жилыми этажами (Kvartiry) и смертельно опасными Коллекторами. Геймплейный смысл этажа: игрок сталкивается с первым серьезным сюжетным препятствием (bottleneck). Лифты вниз заблокированы. Этаж поделен на зоны контроля между двумя фракциями: WILD (слева) и Ликвидаторами (справа), с нейтральной полосой (ничья земля) посередине, где и находятся лифты.

Игрок должен либо помочь одной из фракций и получить пропуск (`outskirts_pass`), либо попытаться прорваться силой, убив лидеров и забрав пропуск с трупа. Атмосфера: полузаброшенные квартиры, баррикады, напряженное противостояние патрулей.

#### Слой `gen/` (Генератор этажа)
- **Файл:** `src/gen/design_floors/outskirts.ts` (НОВЫЙ ФАЙЛ)
Здесь лежит логика процедурной генерации геометрии и политической карты этажа. Генератор должен использовать базовые алгоритмы квартир, но поверх накладывать баррикады и зоны контроля.

```typescript
// TypeScript Stub для src/gen/design_floors/outskirts.ts
import { generateKvartiryBase } from '../procedural_floors/kvartiry';
import { Faction, DoorState } from '../../core/constants';

export function generateOutskirtsFloor(world: World): void {
    // 1. Базовая генерация (коридоры, квартиры)
    generateKvartiryBase(world);
    
    // 2. Распределение территориального контроля (Политическая карта)
    setupFactionZones(world);
    
    // 3. Расстановка баррикад и укреплений
    spawnBarricades(world);
    
    // 4. Обязательная блокировка ВСЕХ лифтов, ведущих ВНИЗ
    lockDownwardLifts(world, 'outskirts_pass');
}

function setupFactionZones(world: World): void {
    // Делим 1024x1024 на 3 вертикальные полосы
    const thirdX = Math.floor(world.width / 3);
    
    for (let x = 0; x < world.width; x++) {
        for (let y = 0; y < world.height; y++) {
            if (x < thirdX) {
                world.factionControl[x][y] = Faction.WILD;
            } else if (x > thirdX * 2) {
                world.factionControl[x][y] = Faction.LIQUIDATOR;
            } else {
                world.factionControl[x][y] = Faction.NEUTRAL;
            }
        }
    }
}

function lockDownwardLifts(world: World, passId: string): void {
    // Ищем клетки лифтов с направлением DOWN
    const downwardLifts = getLifts(world, LiftDirection.DOWN);
    for (const lift of downwardLifts) {
        // Находим двери, примыкающие к шахте лифта
        const doors = getAdjacentDoors(world, lift.x, lift.y);
        for (const door of doors) {
            door.state = DoorState.LOCKED;
            door.keyId = passId; // Без этого ключа дверь не открыть
        }
    }
}
```

### 5. Возможные Edge Cases и Подводные Камни
- **Генерация комнат поверх лифтов:** Убедитесь, что базовая генерация Квартир не перезаписывает важные шахты лифтов из глобального State. Лифты - святое.
- **Пустой этаж:** Если генератор не спавнит НПЦ, убедитесь, что `A-Life materializer` или `population_placement` видит зоны контроля (`world.factionControl`) и спавнит соответствующих патрулей.

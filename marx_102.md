### Шаг 2. Система генерации событий общения (Speech Router)
Создайте систему `speech_router.ts`, которая слушает события мира (через шину `publishEvent`) и решает, кто должен "заговорить".
```typescript
// Заглушка для src/systems/speech_router.ts
export function handleSpeechEvents(world: World, event: GameEvent) {
   if (event.type === 'ENTITY_HURT') {
      // Найти ближайшего НПЦ той же фракции и заставить крикнуть
      const bark = selectBark(world, ['combat', 'ally_hurt'], event.factionId);
      if (bark) emitBark(world, event.bystanderId, bark);
   }
}
```

## Текущая задача
**Звуковое и визуальное оповещение о начале Самосбора**

### Контекст задачи
Событие Самосбор (Samosbor) является ключевой угрозой, но сейчас оно начинается слишком внезапно и без достаточного саспенса. Необходимо реализовать систему поэтапного предупреждения: звук сирены (синтезированный или AudioContext), изменение цвета освещения на багровый, дрожание экрана и вывод текстового предупреждения на всех терминалах этажа.

## Детальный план реализации (Шаги для Агента Jules)

### Шаг 1. Создание фаз Самосбора
Расширьте `src/data/samosbor_director.ts`, добавив фазу `WARNING` перед `ACTIVE`.
```typescript
// Заглушка
export enum SamosborPhase {
  IDLE = 0,
  WARNING = 1,
  ACTIVE = 2,
  AFTERMATH = 3
}
export interface SamosborState {
  phase: SamosborPhase;
  ticksRemaining: number;
}
```

### Шаг 2. Интеграция предупреждающих эффектов (Свет)
В фазе `WARNING` освещение этажа должно плавно менять оттенок на красный (багровый) в `src/render/light.ts`.
Модифицируйте глобальные константы амбиентного света в зависимости от фазы.
```typescript
export function getAmbientLightColor(world: World): Color {
   if (world.samosbor.phase === SamosborPhase.WARNING) {
      const progress = 1 - (world.samosbor.ticksRemaining / MAX_WARNING_TICKS);
      return lerpColor(NORMAL_AMBIENT, RED_AMBIENT, progress);
   }
}
```

### Шаг 4. Взлом терминалов и дверей
Во время `WARNING` и `ACTIVE` фаз гермодвери (hermetic doors) должны переходить в состояние "LOCKED".
Все активные экраны терминалов должны переключиться на красный текст "ВНИМАНИЕ: ПРОТОКОЛ САМОСБОР".
Событие должно рассылаться через `publishEvent`.

### Шаг 5. Интеграция с ИИ НПЦ
НПЦ при наступлении фазы `WARNING` должны бросать текущие дела и бежать в ближайшее укрытие (Shelter). Используйте теги зон (см. `samosbor.md`).

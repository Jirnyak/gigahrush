## Текущая задача
**А-Лайф: Симуляция Перестрелок (динамические бои отрядов).**

### Контекст задачи
В игре должны происходить динамические стычки отрядов. Это не абстрактные цифры, а реальные NPC, использующие боевой ИИ.
- Когда два вражеских отряда встречаются (через систему MacroGoal), начинается перестрелка.
- Звуки выстрелов распространяются на 40 клеток (`publishEvent('gunfire')`).
- Трупы остаются лежать с лутом, мотивируя игрока идти на звук.
- Игрок может обойти зону боя, вмешаться или добить выживших.

### Конкретные файлы и паттерны
- **`src/systems/ai/npc_utility.ts`**: Внедрить override intent `'faction_assault'`, который заставляет NPC двигаться к `target zone`, определенной в `FactionMacroGoal`.
- **Навигация**: Использовать pre-baked nav tree и bounded path chunks, как при обычном патрулировании.
- **События**: `publishEvent('gunfire')` при стрельбе, `publishEvent('zone_captured')` при зачистке территории.

### Детальная спецификация по Архитектуре и Реализации (Строго обязательно к исполнению)

#### Шаг 1: Интеграция Макро-Целей в Микро-ИИ
А-Лайф (макро-уровень) назначает группам NPC цели (`FactionMacroGoal`). Микро-ИИ (utility) должен это учитывать.
```ts
// src/systems/ai/npc_utility.ts
function evaluateFactionAssault(npc: Npc, world: World): UtilityScore {
  // Проверяем, приписан ли NPC к штурмовому отряду
  const squad = world.factions.getSquadByMember(npc.id);
  if (squad && squad.macroGoal === 'assault_zone') {
     // Если мы не в бою, идем к зоне штурма
     if (!npc.aiState.inCombat) {
       npc.aiState.assaultTarget = squad.targetZone;
       return 50; // Высокий приоритет, выше патруля
     }
  }
  return 0;
}
```

#### Шаг 2: Реализация движения отряда (без сверх-накладных расходов)
NPC не нуждаются в жесткой формации. Каждый член отряда самостоятельно строит путь к зоне через `pathfinding.ts` (используя baked nav cache).
```ts
// src/systems/ai/npc_actions.ts
function executeFactionAssault(npc: Npc, world: World) {
  const target = npc.aiState.assaultTarget;
  if (!target) return;
  
  // Стандартная функция steering'а по навмешу
  steerToNavNode(npc, target.center.x, target.center.y, world);
  
  // Если по пути NPC замечает врага (LOS check), стандартная боевая система перехватит контроль (combat intent перевесит assault)
}
```

#### Шаг 3: Звуковая система (Gunfire Events)
При выстрелах (в `src/systems/combat.ts`) обязательно генерируйте событие.
```ts
// src/systems/combat.ts (внутри функции стрельбы)
publishEvent('gunfire', { 
  x: shooter.x, 
  y: shooter.y, 
  volume: weaponDef.loudness || 40 // Дистанция звука в клетках
});
```
Это позволит другим системам (например, спавну мутантов на шум или UI индикаторам) реагировать на бой.

#### Шаг 4: Захват зоны
Система А-Лайф должна периодически (раз в 5 секунд) проверять состояние `MacroGoal`.
Если в `targetZone` нет живых врагов, цель считается выполненной.
```ts
// src/systems/alife/squad_logic.ts
function checkAssaultResolution(squad: Squad, world: World) {
  const enemiesInZone = queryEntitiesInZone(world, squad.targetZone, ...);
  if (enemiesInZone.length === 0) {
    squad.macroGoal = 'defend_zone';
    publishEvent('zone_captured', { factionId: squad.faction, zoneId: squad.targetZone.id });
  }
}
```

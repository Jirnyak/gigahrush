## Текущая задача
**База Ликвидаторов: Геометрия Арены (трибуны, ринг в центре).**

### 1. Глубокий контекст задачи и бизнес-логика
В центре "Базы Ликвидаторов" (дизайн-этаж) должна находиться масштабная арена. Это место, где проходят бои (как НПЦ против НПЦ, так и с участием игрока), делаются ставки и собираются зрители. Арена представляет собой большую комнату, внутри которой выделен огороженный ринг. Вокруг ринга должны быть расставлены стулья/скамейки (трибуны), где могут стоять или сидеть зрители (НПЦ ликвидаторов). Эта комната будет активно переиспользоваться другими системами (системой боев и ставок), поэтому она должна быть четко зарегистрирована в структуре комнаты `world.roomMap`.

### 2. Архитектурные требования и изменения по слоям

        name: 'Арена',
        type: RoomType.COMMON,
        rect: new Rect(startX, startY, ARENA_WIDTH, ARENA_HEIGHT)
    });
    
    // 2. Заливка текстур (стены и пол)
    fillRoomTexture(world, arenaRoom, Tex.F_CONCRETE, Tex.METAL);
    
    // 3. Формирование центрального ринга (внутренний прямоугольник)
    const ringRect = new Rect(startX + 15, startY + 15, 20, 20);
    buildRingPerimeter(world, ringRect);
    
    // 4. Установка трибун (кресла/столы вокруг ринга)
    buildTribunes(world, arenaRoom.rect, ringRect);

    // 6. Регистрация комнаты в мире

    return arenaRoom.id;
}

1. **Создание строителя:** Напишите функцию `buildArenaRoom` со вспомогательными методами для ринга и трибун.

### 5. Возможные Edge Cases и Подводные Камни
- **Пересечение комнат:** Убедитесь, что координаты арены не выходят за границы этажа и не наслаиваются на другие сгенерированные комнаты. Используйте инструменты проверки коллизий комнат.

## Текущая задача
**Арена: Система наград. Титулы, уникальный лут для игрока.**

### 1. Глубокий контекст задачи и бизнес-логика
После успешного прохождения всех волн в режиме выживания на арене (задача marx_17), игрок должен получить ощутимую и уникальную награду. Награда состоит из трех компонентов:
1. **Материальная:** Редкий предмет-трофей (например, Золотой Кубок Арены), который стоит много денег и служит символом статуса.
2. **Социальная (Репутация):** Массовое повышение уважения (relation) со стороны всех ликвидаторов на этаже (Bulk Update). Они должны начать здороваться по-другому или игнорировать мелкие нарушения.
3. **RPG (Опыт):** Огромная разовая порция XP за завершение глобального квеста-испытания.

#### Слой `systems/` (Системы и Логика Наград)
- **Файл:** `src/systems/reputation.ts` и модуль RPG
Создайте функцию, которая объединяет все компоненты выдачи награды. Слушайте событие `ARENA_SURVIVAL_WON`.

```typescript
// TypeScript Stub для src/systems/arena_rewards.ts
import { getEntitiesByFaction } from './entity_index';
import { addXp } from './rpg';
import { createItem } from './inventory';

export function grantArenaChampionRewards(world: World, player: Entity): void {
    // 1. Выдача уникального предмета прямо в инвентарь (или спавн сундука перед игроком)
    const trophy = createItem('arena_gold_trophy');
    player.inventory.push(trophy);
    
    // 2. Массовое улучшение репутации (Bulk Update)
    // Находим всех ликвидаторов на активном этаже
    const liquidators = getEntitiesByFaction(world, Faction.LIQUIDATOR);
    for (const npc of liquidators) {
        // Добавляем 30-50 пунктов к личному отношению к игроку
        npc.playerRelation = Math.min(100, npc.playerRelation + 30); 
    }
    
    // 3. Выдача бонусного опыта
    addXp(player, 1500); // Крупная сумма
    
    // 4. Публикация глобального события (чтобы система слухов (Rumors) подхватила это)
    publishEvent('arena_champion_crowned', { playerId: player.id });
}
```

- **Интеграция с системой слухов (`src/systems/rumor.ts`):**
Событие `'arena_champion_crowned'` должно генерировать слух. НПЦ в диалогах (Bark) могут начать произносить фразы вроде "Видели этого психа на ринге? Уважуха."

### 5. Возможные Edge Cases и Подводные Камни
- **Переполненный инвентарь:** Если инвентарь игрока имеет ограничение по весу (overweight limits) или слотам, кубок может не поместиться. Рассмотрите возможность спавна кубка физически на земле перед игроком (`placeItemOnGround()`), если `inventory.push` не удался.
- **Абуз (Эксплойт):** Событие победы не должно вызываться дважды за один бой (race condition). Убедитесь, что стейт арены очищается сразу ПЕРЕД выдачей награды.

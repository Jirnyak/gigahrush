Каждая зона имеет `ownerFactionId`, `defenseLevel`, `resourceValue`.

  roomIds: number[];

  defenseScore: number;

### Шаг 2. Разработка системы Factions War (Глобальный ИИ)
Создайте мета-систему в `src/systems/factions_war.ts`, которая вызывается раз в 1-5 минут игрового времени (а не каждый кадр).
Фракции оценивают соседние зоны. Если `attacker.power > defender.defenseScore + threshold`, происходит атака.
```typescript
// Заглушка
export function simulateFactionWars(world: World) {
  for (const faction of world.factions) {
     // Выбор цели для расширения
     const targetZone = findWeakestAdjacentZone(world, faction);
     if (targetZone) {
        launchAssault(world, faction.id, targetZone.id);
     }
  }
}
```

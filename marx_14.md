## Текущая задача
**База Ликвидаторов: Симуляция 'За периметром' (дикие зоны, перестрелки).**

### 1. Глубокий контекст задачи и бизнес-логика
Этаж "База Ликвидаторов" — это не просто безопасная зона, это укрепленный аванпост в опасном мире. За периметром базы (за стенами и гермодверями) лежат так называемые "Дикие зоны" (Wild zones). В этих зонах должны спавниться дикие мобы, мутанты и бандиты. 

Суть задачи: создать живую, не заскриптованную симуляцию конфликта. Ликвидаторы отправляют патрули за периметр, где они сталкиваются с дикими врагами. Бой (перестрелки) должен происходить естественным образом за счет существующих систем AI и территориальной враждебности, без использования runtime-спавнеров или жестких костылей. Игрок сможет наблюдать эти стычки, помогать патрулям или мародерить трупы.

### 2. Архитектурные требования и изменения по слоям
Ваша задача интегрирует слои генерации, данных фракций и AI utility.

Вам необходимо добавить AI Intent для патрулирования периметра. Выберите несколько НПЦ-ликвидаторов, назначьте им intent `PATROL`, задайте маршрут, проходящий через внешние коридоры, прилегающие к "диким зонам".

import { Intent, AggroLevel } from '../../core/ai_types';

export function setupLiquidatorPerimeterPatrols(world: World): void {
    const liquidators = getEntitiesByFaction(world, Faction.LIQUIDATOR);
    
    // Выбираем 3-5 НПЦ для внешнего патруля
    const patrolGroup = selectPatrolGroup(liquidators, 5);
    
    // Генерация массива путевых точек вокруг базы
    const perimeterRoute = generatePerimeterWaypoints(world);
    
    for (const npc of patrolGroup) {
        assignIntent(npc, Intent.PATROL, {
            route: perimeterRoute,
            aggressiveness: AggroLevel.HIGH, // Будут атаковать при виде врага
            returnToBaseHp: 30 // Возврат на базу при низком HP
        });
    }
}
```

export function populateWildZones(world: World): void {
    const wildSpawnPoints = findOuterPerimeterCells(world);
    const SPAWN_COUNT = 8;
    
    for (let i = 0; i < SPAWN_COUNT; i++) {
        if (wildSpawnPoints.length === 0) break;
        
        const pt = popRandom(wildSpawnPoints);
        const wildMob = spawnMonster(world, 'bandit_scavenger'); // или мутант
        
        // Даем им оружие
        wildMob.inventory.push(createItem('rusty_pistol'));
        wildMob.weapon = 'rusty_pistol';
        
        placeEntity(world, wildMob, pt.x, pt.y);
    }
}
```

### 3. Пошаговый план внедрения (Action Plan)
1. **Модификация Генерации:** В `liquidator_base.ts` добавьте функцию одноразового спавна мобов во внешних коридорах/зонах.
2. **Настройка AI Патрулей:** В конце генерации или на этапе `A-Life materialization` выберите часть сгенерированных Ликвидаторов и дайте им `Intent.PATROL`.
3. **Построение Маршрута:** Реализуйте алгоритм `generatePerimeterWaypoints`, который находит ключевые перекрестки вокруг базы и связывает их в маршрут патрулирования.
4. **Проверка враждебности:** Убедитесь в `factions.ts`, что Ликвидаторы и Wild имеют статус врагов, чтобы стандартная боевая система (combat AI) сработала, когда они увидят друг друга.
5. **Запрет Runtime Спавнеров:** ВАЖНО: Не пишите таймеры (`setInterval`), которые бесконечно спавнят мобов из воздуха. Если патруль убил всех диких — зона зачищена до следующего Самосбора.

   - Вы должны увидеть, как патруль Ликвидаторов движется по маршруту.
   - Когда они замечают мобов из Wild Zone, они должны автоматически вступить в перестрелку.
   - Проверьте, что оружие стреляет, HP отнимается, и трупы падают с лутом.

### 5. Возможные Edge Cases и Подводные Камни
- **Лимиты Entity:** Убедитесь, что спавн диких мобов не превышает жесткие глобальные лимиты (`world.entityLimits`). Используйте проверку `checkEntityLimit()` перед спавном.
- **Глупый ИИ:** Патрульные могут застрять в узких коридорах, пытаясь расстрелять моба за стеной. Опирайтесь на существующий `combat_ai.ts`, который использует LOS (Line of Sight) перед выстрелом.

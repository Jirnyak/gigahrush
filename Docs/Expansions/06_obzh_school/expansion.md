# Expansion 06: Школа ОБЖ имени гермодвери

Версия: 0.1 design  
Статус: технический дизайн для разработки  
Роль в линейке: социальный survival expansion и поздний обучающий слой  
Базовые источники: `README.md`, `desdoc.md` разделы SCHOOL, быт, A-Life, квесты, самосбор

## Назначение

«Школа ОБЖ имени гермодвери» добавляет образовательный блок, где детей учат переживать самосбор как обычную пожарную тревогу. Это не веселая школьная зона и не стартовый tutorial. Это жесткий социальный экспаншн про систему, которая делает катастрофу частью расписания: уроки герметизации, кабинет труда, радиокружок, спортзал-убежище, журнал эвакуации и дети, которые знают правила лучше взрослых.

Экспаншн дает игроку новые навыки, но цена этих навыков - ответственность за слабых NPC и моральные решения во время тревоги.

## Игровая задача

В игре уже есть актовый зал и обучающие элементы через Ольгу, Барни и Якова. Школа не заменяет их. Она становится mid-game контентом, который учит более сложным системам: эвакуации, гермодверям, ремонту, радио, документам, панике NPC и предвестникам самосбора. Игрок видит, как гигахрущ воспроизводит себя через детей.

Основной вопрос: кого ты успеешь закрыть за дверью, если сирена началась посреди урока?

## Основная игровая петля

Игрок попадает в школьный блок по просьбе учителя, родителя или ликвидатора. В обычном режиме он помогает с уроками: чинит гермодверь, приносит мел, настраивает радио, достает пайки, проверяет убежище. Затем происходит тревога или ложная тревога. Игрок должен применить подготовленные решения: направить детей, открыть/закрыть двери, выбрать короткий или безопасный маршрут, удержать панику, решить, кого спасать первым.

После события школа меняется. Дети помнят, кто помог. Учителя дают доступ к навыкам или документам. Родители требуют объяснений. Министерство требует отчет, даже если отчет невозможен.

## Места

MVP размещается как крупный POI в `LIVING` или отдельный school pocket:

| Место | Функция |
| --- | --- |
| Кабинет ОБЖ | правила, плакаты, учебные перки |
| Кабинет труда | простой крафт и ремонт дверей |
| Радиокружок | слухи, предвестники, связь с метро/556 |
| Спортзал-убежище | эвакуационная arena |
| Столовая | пайки, конфликт с грибами/рынком |
| Учительская | документы, ключи, моральные решения |
| Подвал инвентаря | риск, монстр, контрабанда |

Полная версия может добавить детсад без окон, школьный двор без неба и «класс, которого нет в расписании».

## Учебные навыки

Школа может выдавать micro-perks через уроки, но они должны быть маленькими:

| Навык | Эффект |
| --- | --- |
| «Слушай сирену» | чуть раньше HUD-предвестник тревоги |
| «Дверь на себя» | быстрее взаимодействие с гермодверью |
| «Не беги в туман» | меньше panic/fear при fog |
| «Кружок пайки» | простой ремонт/крафт с меньшим шансом поломки |
| «Радиолинейка» | лучше распознавание ложных объявлений |

Навыки не должны превращать игрока в супергероя. Это бытовые преимущества за участие в школьных событиях.

## Эвакуация как система

Эвакуация работает через небольшие группы NPC, а не индивидуальный pathfinding каждого ребенка каждую frame. Группа имеет цель, panic, teacherAlive, routeState и count. Игрок управляет дверями, маршрутами и ключевыми NPC, а не каждой фигуркой.

```ts
export interface EvacGroupState {
  id: string;
  count: number;
  panic: number;
  currentRoomId: number;
  targetRoomId: number;
  routeState: 'waiting' | 'moving' | 'blocked' | 'sealed' | 'lost';
  lastTick: number;
}
```

Tick идет редко и только для активного события. Это сохраняет масштаб без дорогого crowd simulation.

## Самосбор

Школьный самосбор страшен тем, что правила уже написаны на стенах, но ситуация все равно ломается. Классический вариант проверяет базовую эвакуацию. Тихий вариант начинается без сирены, и радиокружок становится важнее плакатов. Мокрый вариант затапливает спортзал или столовую. Электрический гасит лампы и радио. Мясной резонанс делает один класс «непригодным к учету», как будто его никогда не было.

Школа должна показывать предвестники лучше других зон, потому что дети тренированы. Но лучшее предупреждение не гарантирует спасение.

## NPC и сюжет

Ключевые NPC:

| NPC | Роль |
| --- | --- |
| Нина ОБЖ | учитель, жесткая и усталая, знает реальные протоколы |
| Паша Радиокружок | ученик, ловит чужие этажи |
| Завхоз Мелихов | ключи, ремонт, закрытые подсобки |
| Мать из очереди | конфликт: забрать ребенка сейчас или ждать приказ |
| Ликвидатор-инструктор | превращает тренировку в зачистку |

Сюжетная арка не должна делать детей расходным ресурсом ради шока. Последствия должны быть серьезными, но игра не должна смаковать смерть. Фокус: ответственность, бюрократия, страх и память NPC.

## Техническая интеграция

Планируемые файлы:

| Файл | Назначение |
| --- | --- |
| `src/gen/living/obzh_school.ts` | школьный POI |
| `src/data/school_lessons.ts` | уроки, micro-perks, проверочные события |
| `src/systems/evacuation.ts` | группы эвакуации и редкие ticks |
| `src/data/dialogue.ts` | учителя, дети, родители |
| `src/data/notes.ts` | плакаты ОБЖ, журналы эвакуации, расписание |
| `src/systems/debug.ts` | начать учебную тревогу, форсировать тихий вариант |

Если глобальная система events уже есть, школьные тревоги регистрируются как event defs. Если нет, первый срез запускается через квест/диалог/debug.

## Производительность и Math LOD

Low: одна школа, одна группа эвакуации, минимум pathing, события scripted.

Middle: 2-3 группы, panic, блокировки дверей, micro-perks.

High: родители и фракции реагируют после эвакуации, радиокружок связывается с метро/номерными этажами, документы меняют исход.

Ultra: плотный школьный визуал: плакаты, парты, спортзал, толпа через sprites и звук. Логика остается групповой.

## Вертикальный срез

MVP готов, когда игрок может пройти один школьный prep-квест, получить малый навык, затем пережить тревогу с групповой эвакуацией и увидеть последствия результата в диалоге/документе.

Definition of Done:

| Область | Проверка |
| --- | --- |
| Обучение | минимум 3 micro-perks, один реально применяется |
| Эвакуация | одна группа NPC имеет states waiting/moving/blocked/sealed |
| Самосбор | минимум 2 варианта меняют условия тревоги |
| Контент | учитель, ученик радиокружка, завхоз, родитель |
| Последствия | результат события записан в квест/журнал/диалог |

## Риски

Главный риск: школа станет вторым tutorial и потеряет horror. Контрмера: разместить ее после базового обучения, привязать к эвакуации и ответственности.

Второй риск: массовые NPC сломают производительность. Контрмера: группы вместо individual AI, редкие ticks, ограниченная arena.

Третий риск: тон станет эксплуатационным. Контрмера: не смаковать страдание детей; показывать систему, взрослых, документы и последствия.

## Связь с другими экспаншнами

Школа использует документы Райсовета, пайки Грибной смены, слухи метро, лекарства Больничного блока и ремонт Теплотрассы. Она дает игроку бытовые навыки, которые потом применяются в более опасных DLC.

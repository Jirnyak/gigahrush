## Текущая задача
**Фикс наложения спрайтов лута (z-index) и интерфейс кучи предметов**

### Контекст задачи
При генерации кучи предметов (например, из разрушенного контейнера или убитого врага) спрайты накладываются друг на друга хаотично. Z-index может ломаться, и игроку тяжело поднять нужный предмет кнопкой E. Нужно ввести интерфейс "Лут рядом" (Nearby Loot Area) или меню контейнера, а спрайты красиво раскидывать (Physics Scatter) с правильной сортировкой глубины.

### Шаг 1. Разброс лута (Physics Scatter)
Вместо спавна предметов в точных координатах x,y добавьте им начальный вектор скорости (velocity) при выпадении, чтобы они разлетались в радиусе 1-2 тайлов.
```typescript
// Заглушка для src/systems/items.ts
export function spawnLootExplosion(world: World, x: number, y: number, items: ItemDef[]) {
  items.forEach(item => {
    const angle = Math.random() * Math.PI * 2;
    const speed = 1 + Math.random() * 2;
    const entity = createItemEntity(world, item, x, y);
    entity.vx = Math.cos(angle) * speed;
    entity.vy = Math.sin(angle) * speed;
  });
}
```

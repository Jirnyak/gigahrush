## Текущая задача
**Туториал: Поиск ключа и выход (финал обучения).**

### Контекст задачи
Вместо скриптового Самосбора, финалом туториала является квест на нахождение ключа от входной двери квартиры. Игрок должен исследовать комнаты, найти ключ (или получить его от соседа) и открыть замок. Это действие знаменует конец туториала. 

*Примечание к раннему плану*: Прежняя версия документа упоминала "Step SAMOSBOR" и таймер. Мы интегрируем этот таймер как атмосферный таймер давления (мягкий дедлайн), чтобы стимулировать игрока покинуть уровень, но ключевая механика финала — это именно дверь и ключ, а не Самосбор, стирающий уровень.

### Конкретные файлы и паттерны
- **`src/systems/tutorial.ts`**: Step `SAMOSBOR` заменяется или дополняется шагами `FIND_KEY` и `UNLOCK_DOOR`. Нужно интегрировать вызов завершения туториала и снятия ограничений.
- **Route cue**: Через `src/systems/route_cues.ts` — маркер на гермодверь и на примерную локацию ключа.
- **Гермодверь**: Одна на этаже, закрыта на замок (`locked = true`). За ней — переход в основную игру.
- **Таймер**: Если игрок не успеет покинуть квартиру за N секунд после старта финала туториала — атмосферное давление (может тикать -HP от токсичного тумана, нагнетаемого снаружи).

#### 3. Системы и логика (Слой `systems/`)
Добавьте обработку взаимодействия с запертой дверью и проверку наличия ключа в инвентаре. Вся логика должна находиться в `src/systems/`.

```typescript
// В файле src/systems/tutorial_logic.ts или src/systems/interactions.ts

import { TutorialStep } from '../data/tutorial_steps';
import { ITEM_TUTORIAL_KEY } from '../data/items';

export function handleTutorialDoorInteraction(world: World, player: Entity, door: Door): InteractionResult {
    // Проверяем, есть ли у игрока нужный ключ
    const hasKey = player.inventory.some(item => item.id === door.requiredKeyId);
    
    if (door.isTutorialExit) {
        if (hasKey) {
            door.locked = false;
            publishEvent({ type: 'TUTORIAL_DOOR_UNLOCKED' });
            advanceTutorial(world, TutorialStep.EXIT_APARTMENT);
            return { success: true, message: 'Дверь со скрипом поддалась. Путь свободен.' };
        } else {
            return { success: false, message: 'Заперто намертво. Нужно найти ключ.' };
        }
    }
    return { success: false, message: 'Дверь не поддается.' };
}

// Слушатель событий подбора предметов
export function onTutorialItemPickedUp(world: World, item: Item) {
    if (item.id === ITEM_TUTORIAL_KEY.id && world.tutorialStep === TutorialStep.FIND_KEY) {
        advanceTutorial(world, TutorialStep.UNLOCK_DOOR);
        publishEvent({ type: 'UI_MESSAGE', text: 'Ключ найден. Откройте входную дверь.' });
    }
}
```

#### 4. Мягкий таймер давления (Стимул к выходу)
Интегрируйте таймер, упомянутый в раннем описании задачи. Если игрок не выходит, атмосфера должна нагнетаться:

```typescript
// В src/systems/tutorial_timer.ts
export function updateTutorialPressure(world: World, dt: number) {
    if (world.tutorialStep === TutorialStep.UNLOCK_DOOR || world.tutorialStep === TutorialStep.FIND_KEY) {
        world.tutorialExitTimer += dt;
        
        // Включаем тревожные эффекты
        if (world.tutorialExitTimer === 30) {
            publishEvent({ type: 'PLAY_SOUND', soundId: 'distant_rumble' });
            publishEvent({ type: 'UI_MESSAGE', text: 'В коридорах что-то гудит. Надо спешить.' });
        }
        
        // Начинаем наносить периодический урон (имитация тумана)
        if (world.tutorialExitTimer > 60 && world.tutorialExitTimer % 1 === 0) {
            applyDamage(world.player, 1, 'tutorial_fog');
            // Визуальный эффект экрана
            publishEvent({ type: 'RENDER_SCREEN_EFFECT', effect: 'vignette_red' });
        }
    }
}
```
